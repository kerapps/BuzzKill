## Extension Name

LinkedOut — LinkedIn Corporate Speak Translator

## Short Description (132 chars max)

Translate LinkedIn corporate speak into plain language, hide promoted posts, and write corporate-style posts from plain text.

## Detailed Description

LinkedOut helps you cut through LinkedIn jargon.

Features:

- Translate feed posts into plain language
- Choose translation tone (blunt, sarcastic, neutral)
- Hide promoted posts
- Optional "Hide original" mode
- In-page LinkedOut HUD with quick settings
- "Create a post" mode that converts plain text into LinkedIn-style corporate speak
- Local cache to reduce repeated API costs

How it works:

- You provide your own API key (OpenAI or Anthropic)
- LinkedOut sends post text only when translation is requested
- Translations and settings are stored locally in browser storage

Data handling:

- No custom backend
- No selling personal data
- Data is sent only to the provider you selected
